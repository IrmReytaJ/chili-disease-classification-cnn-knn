# Klasifikasi Penyakit Cabai Menggunakan CNN dan KNN

penelitian ini diterapkan metode Convolutional Neural Network (CNN) sebagai model utama dan K-Nearest Neighbor (KNN) sebagai metode pembanding untuk menganalisis perbedaan performa dalam klasifikasi gambar cabai.

## Hasil
Berdasarkan hasil pengujian, metode CNN memiliki tingkat akurasi yang lebih tinggi dibandingkan metode KNN dalam melakukan klasifikasi citra cabai.
